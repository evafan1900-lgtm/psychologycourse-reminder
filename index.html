课程知识要点

Lesson 9: 团队流程

  • • 社会惰化：团队成员减少个人贡献
  • • 团队凝聚力：成员间的人际纽带
  • • 团队角色：明确责任分工
  • • 团队效能：整体大于部分之和

Lesson 10: 沟通

  • • 三要素：发送者、接收者、信息
  • • 沟通网络：集中式vs分散式
  • • 沟通氛围：支持性vs消极性
  • • 信任作用：沟通的基石

Lesson 11: 合作竞争

  • • 混合动机：合作与竞争并存
  • • 竞争积极面：创新、效率、动力
  • • 竞争消极面：目标混乱、不信任
  • • 合作优势：信任、协调、共赢

Lesson 12: 冲突管理

  • • 冲突类型：个人/实质/程序冲突
  • • 解决方法：对峙/合作/妥协/规避
  • • 谈判技巧：分配式vs整合式
  • • 积极作用：促进创新和思辨

Lesson 13: 虚拟团队

优势：

  • • 节约时间和成本
  • • 扩大人才库
  • • 灵活性强

劣势：

  • • 沟通不畅风险
  • • 信任建立困难
  • • 技术依赖性强

踩分攻略

第二周作业写作要点

1. 团队评估框架：

  • • 合作水平：团队成员协作程度、信息共享情况
  • • 竞争态势：内部竞争vs外部竞争，积极竞争vs恶性竞争
  • • 冲突状况：冲突类型识别、解决机制评估

2. 解决方案建议：

  • • 运用课程理论支撑你的分析
  • • 提出具体可行的改进措施
  • • 结合实际案例说明效果

期中作业写作指南

Individual Process Paper (1-2页)：

  • • 团队目标分析：明确性、一致性、可达性
  • • 角色分工评估：角色清晰度、权责匹配
  • • 规范建立情况：明文规定vs隐性约定
  • • 领导力体现：领导风格、影响力分析
  • • 沟通效果：信息流畅度、反馈机制
  • • 合作协调：团队协作水平、冲突处理

🏆 高分技巧：

  • • 理论与实践结合，引用课程概念
  • • 具体案例支撑，避免空泛描述
  • • 提出改进建议，展现批判性思维
  • • Word格式，1-2页篇幅控制好
